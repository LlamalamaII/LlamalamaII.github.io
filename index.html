Abstract

We present an intelligent English teaching application that leverages Large Language Models (LLMs) with Retrieval-Augmented Generation (RAG) and low-level PyTorch-based cosine similarity computation to enhance language instruction. Our system allows users to upload PDF documents, which are transformed into a vector database for semantic search and contextualized dialogue. The teaching assistant engages users through text-to-speech (TTS) synthesis and real-time voice conversion using RVC, enabling natural and interactive conversation. We evaluate the systemâ€™s effectiveness using standardized English proficiency benchmarks, including M6-ENG and IELTS writing assessments. Furthermore, we conduct pre-test and post-test evaluations with real learners, complemented by user questionnaires to assess engagement and learning gains. Our results suggest the system provides meaningful improvements in language acquisition and offers a practical framework for LLM-based educational tools.
